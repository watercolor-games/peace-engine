ListStream
A Stream interface to an internal List of bytes. Reading will pop bytes from the top and writing will push them to the bottom. This is like a UNIX pipe in a way. One end of the pipe should see this as read-only and the other as write-only, which you can accomplish with a ReadOnlyStream and WriteOnlyStream around this one.
ReadOnlyStream
This is a wrapper to make another Stream read-only.
WriteOnlyStream
This is a wrapper to make another Stream write-only.
